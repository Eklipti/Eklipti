## Примеры использования

[Конвертор EFKI](https://github.com/Eklipti/inbg-conventor-xml-for-bki) - Утилита для обработки и конвертации данных о 
кредитных субъектах и событиях из Excel-файлов в специализированные XML-форматы для различных 
Бюро кредитных историй (ОКБ, Скоринг, КБРС, НБКИ).

[Парсер ЕФРСБ/КАД](https://github.com/Eklipti/fedresurs-kad-parser) - Асинхронный скрипт для массового сбора данных 
о банкротстве физических и юридических лиц. Более адекватная и стабильная версия является приватной и 
недоступна на Github _(временно)_.

[Telegram Chat Analyzer](https://github.com/Eklipti/telegram-chat-analyzer) - Инструмент (ETL) для 
анализа чатов Telegram на основе JSON-экспорта из Desktop приложения. 
**В 9 раз больше метрик** и **в 3 раза больше инструментария**, чем у типичных анализаторов 
_(согласно моей обывательской статистики)_.

[Telegram Control Bot](https://github.com/Eklipti/TelegramControlBot) - управляйте собственной Windows 10+
через личного бота в Telegram. Проект не стабилен, давно не обновлялся _(из-за ограничений и обучения 
настройки прокси)_.
